# AI Lane Monitoring System

Advanced driver assistance system for real-time lane departure detection and road safety alerts.

## Features

- Real-time lane line detection using Hough transforms
- Lane departure warning with audio alerts
- Region-of-interest based processing for efficiency
- Video feed and dashcam input support
- Lane curvature estimation

## Tech Stack

Python, OpenCV, NumPy, Deep Learning

## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
git clone https://github.com/karthik-idikuda/AI-Lane-Monitering-System.git
cd AI-Lane-Monitering-System
pip install -r requirements.txt
```

### Usage

```bash
python lane_detect.py --source video.mp4
```
